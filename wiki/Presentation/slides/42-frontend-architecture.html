How the Website Feels Fast

The app keeps Django as the dependable server-rendered foundation, then layers vanilla JavaScript on top so navigation, modals, live updates, and theme changes feel smooth without adopting a frontend framework.

Django pages Real server-rendered pages stay as the foundation
Vanilla JS Enhances navigation, modals, and live updates
Partial fetch Navigation requests only the main page fragment
Content swap The browser replaces the page body without a full reload
Scripts + theme Page scripts restart while light, dark, or system theme persists
