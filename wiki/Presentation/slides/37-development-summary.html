What Web Development Built

The web app gives the lab a single place to log in, request templates, watch builds, and start VMs without asking users to work directly inside Proxmox.

LoginActive Directory sign-in creates the user session and determines template access.
Template wizardFaculty can choose an OS, ISO, software, hardware, and network settings.
Build trackingLong Packer builds run in the background while the UI shows progress.
VM startUsers can provision a VM from one of their completed templates.
