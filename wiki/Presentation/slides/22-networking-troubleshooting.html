Networking / Troubleshooting

Networking Troubleshooting

Issue Virtual machines could not communicate with other virtual machines.
Solution Applied the correct VLAN tag or corrected the VM's misconfigured IP address.
Issue Virtual machines changed IP addresses or had the wrong IP assigned.
Solution Updated and saved the IP configuration in the correct system file.
Issue Amy and Famsworth servers shut down because of configuration issues.
Solution Restored the affected physical switch interfaces to the previous known-good configuration.
Issue Switch interfaces were down or assigned to the wrong native VLAN.
Solution Applied the proper native VLAN and used no shutdown where needed.
