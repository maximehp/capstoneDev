Monitoring / Grafana

Grafana: Dashboards & Alerts

  • Open-source visualization platform — turns raw metrics into dashboards, charts, and graphs.
  • Installed on Ubuntu Server 24.04.3 on Amy; accessible at http://172.27.85.4:3000.
  • Prometheus is the primary data source; OpenSearch provides indexed log data from Graylog.
  • Dashboards built with PromQL — metrics written in query code directly due to builder limitations.
  • Alerts trigger when metrics cross thresholds; availability alerts include a reboot delay to reduce noise.
  • Alert notifications route to Slack.
