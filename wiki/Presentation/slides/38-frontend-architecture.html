How the Website Feels Fast

  • Django still renders the real pages, so the app works from a simple server-side foundation.
  • Vanilla JavaScript adds smoother navigation, modal behavior, and live updates.
  • Only the main page content is swapped during in-app navigation, so moving around feels quicker.
  • The same layout supports light, dark, and system theme preferences.
