Frontend Architecture

  • Server-rendered Django templates are the source of truth for page HTML.
  • Frontend uses vanilla JavaScript and CSS with no framework.
  • Partial navigation uses fetch requests with X-Requested-With headers.
  • Page updates replace #app-content and page-specific assets dynamically.
  • Theme system supports light, dark, and system modes.
