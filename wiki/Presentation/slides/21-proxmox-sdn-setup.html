Proxmox Setup and Configuration

Proxmox networking was extended with Open vSwitch bonds and bridges so VMs could land in the correct subnet group.

  • Selected bond modes based on the role of each network path.
  • Entered static IP addresses for each virtual machine.
  • Used VLAN tags on VMs for proper subnet connectivity.
