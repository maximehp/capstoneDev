MinIO

  • MinIO is a self-hosted S3-compatible object store running on TrueNAS.
  • Stores Terraform state files as the remote backend.
  • Stores vm_info.json for Ansible to consume after provisioning.
  • Decouples Terraform and Ansible so they do not need to run in the same job.
