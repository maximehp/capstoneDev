Infrastructure - IaC / Pipeline

Packer

Builds a pre-configured VM template cloned by Terraform — ISO boots the VM, Packer provisions over SSH, hands off to Ansible, then snapshots to Proxmox.

Boot Phase ISO boots the VM — CDROM delivers boot commands and writes the Packer SSH key to authorized_keys
Packer SSH In & Provision Packer connects via SSH, runs shell scripts, installs apt packages — Docker, qemu-guest-agent, and common agents
Key Handoff Ansible SSH key injected into authorized_keys — Packer key removed, leaving a clean handoff
Golden Template VM shut down and snapshotted — stored in Proxmox as a template for Terraform to clone
