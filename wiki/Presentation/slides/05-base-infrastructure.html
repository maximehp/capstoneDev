Infrastructure / Systems Overview

Base Infrastructure

  • Inherited a rack environment at the start of the semester.
  • Core networking provided by two Layer 3 switches named Bessie.
  • Proxmox cluster used Amy and Farnsworth as controllers with seven compute hosts: Fry, Leela, Bender, Hermes, Zoidberg, Scruffy, and Nibbler.
  • Kif operated separately as a dedicated build and development node.
  • Three TrueNAS systems - HypnoToad, Calculon, and Mom - provided pipeline storage.
  • VMID ranges were assigned per group to prevent provisioning conflicts.
